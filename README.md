# Clothing Brand Website
A modern and responsive website showcasing a clothing brand's collection with smooth animations and interactive features.

## Description
This project is a responsive website built with HTML, CSS, and JavaScript. 
It displays various clothing items, including sweaters, skirts, hoodies, and one-piece outfits, with engaging animations and an interactive user experience. The site is optimized for both desktop and mobile devices.

## Features
- Responsive design for mobile and desktop
- Cat animation following the mouse (fun interactive element)
- Product gallery with hover effects
- Smooth scrolling navigation
- Swiper.js carousel for featured products or testimonials
- Modern and clean layout with CSS animations
